# logo-01-black

Glue WordPress para o isotipo `/v7`.

**Cena Three.js:** `src/three/FractoLogoScene.ts` + `src/lib/fractoLogoConfig.ts` (fonte única).

Este package só contém:

- `src/index.ts` — mount, autoInit, `data-fracto-3d`
- `src/embed.css` — estilos do embed no WP

## Build

```bash
npm run build:wp          # na raiz do repo
```

Saída: `../../themes/Fracto/assets/3d/logo-01-black/`
