# background-grid-black

Glue WordPress para o grid do hero `/v5`.

**Cena Three.js:** `src/three/instancedGridSceneV5.ts` (fonte única — não duplicar aqui).

Este package só contém:

- `src/index.ts` — mount, autoInit, `data-fracto-3d`
- `src/embed.css` — estilos do embed no WP

## Build

```bash
npm run build:wp          # na raiz do repo
```

Saída: `../../themes/Fracto/assets/3d/background-grid-black/`
