# Assets 3D — gerados automaticamente

Esta pasta é preenchida pelo build. **Não edite manualmente.**

```bash
# Na raiz do repo Vue (Fracto/):
npm run build:wp
```

Saída:

```text
assets/3d/
├── background-grid-black/   ← hero /v5
│   ├── background-grid-black.css
│   ├── background-grid-black.min.js
│   └── background-grid-black.min.js.map
└── logo-01-black/           ← isotipo /v7
    ├── logo-01-black.css
    ├── logo-01-black.min.js
    └── logo-01-black.min.js.map
```

No Hostinger: copie a pasta `wordpress/themes/Fracto/` inteira para `wp-content/themes/Fracto/`.
